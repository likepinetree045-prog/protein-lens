## 3. ⭐ 검증 — 이 화면만 보면 됩니다

배포 후 폰 주소창에:

```
https://<당신-도메인>.vercel.app/api/debug?secret=<DEBUG_SECRET>
```

화면 JSON에서 **이 3개만** 확인:

| 봐야 할 것 | 정상이면 | 아니면 볼 곳 |
|---|---|---|
| `"allOk"` | `true` | 아래 두 줄 확인 |
| `smoke` → `gemini` → `ok` | `true` | 4단계 `GEMINI_API_KEY` / 재배포 / `limit`이면 `GEMINI_MODEL` 교체 |
| `smoke` → `db` → `ok` | `true` (`테이블 2/2`) | 2단계 Neon 연결 / 재배포 |

> ⚠️ 테이블은 **직접 안 만듭니다.** 앱이 자동 생성합니다. SQL 에디터 만지지 마세요.

## 4. 친구에게 주기 전 30초 셀프 테스트 (폰에서)

- [ ] 루트 링크 열기 → 달력이 뜨는가

- [ ] **오늘 날짜 탭** → **사진으로 추가** → 음식 사진 1장
- [ ] 단백질 숫자(히어로) + 이름 + 정확/추정 뱃지가 뜨는가
- [ ] **이 날짜에 저장** → 달력 그 날 칸에 누적 g가 표시되는가

여기까지 되면 → **친구에게 루트 링크 전송 = 끝.**
